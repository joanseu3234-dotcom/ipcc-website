🚀 一鍵部署設定

設定完成後，每次在後台按「🚀 發布上線」即可自動更新官網，不需要開任何程式。

第 ① 步 — GitHub Token
每台電腦需輸入一次，Token 只存在此瀏覽器中，不會上傳。
儲存後自動同步到所有電腦，只需填一次。
沒有 Token？→ 點此建立 GitHub Token （打開頁面後直接拉到底點「Generate token」，記得勾選 repo）
✅ 已設定 GitHub Actions 自動部署：每次按 🚀 推送到 GitHub，Zeabur 就會自動重新部署，不需要其他設定。
📁 首次設定 — 上傳所有網站檔案到 GitHub
第一次使用時，點此按鈕選擇桌面的 IPCC 資料夾，所有檔案將自動推送到 GitHub。
完成後，之後每次修改內容只需按 🚀 發布上線，網站就會自動更新。

⚠️ 危險區域

清除所有資料後無法復原，請謹慎操作。
