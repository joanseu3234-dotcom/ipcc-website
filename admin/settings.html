🚀 發布上線設定（任何電腦都能用）

第 ① 步 — GitHub 設定（每台電腦需輸入一次 Token）
此設定儲存後會自動同步到所有電腦，只需填一次。
Token 只存此瀏覽器，不公開。 點此建立 Token（勾選 repo 權限）
第 ② 步 — Zeabur Deploy Hook（只需設定一次，全電腦共用）
設定後自動嵌入部署檔案，任何電腦無需重設。
如何取得 Deploy Hook URL：
1. 開啟 dash.zeabur.com
2. 進入你的 Project → 點選 ipcc-website 服務
3. 上方點「Settings」→ 找到「Deploy Hook」→ 點「Create」
4. 複製產生的 URL 貼到上方欄位
第 ③ 步 — 連結 Zeabur 與 GitHub（只需做一次）
1. Zeabur Dashboard → 你的服務 → Settings → Git Source
2. 點「Connect GitHub」→ 選擇 joanseu3234-dotcom/ipcc-website
3. 選 Branch: main → Save
完成後，🚀 按下時 GitHub 有新內容，Zeabur 即自動重新部署。
📁 首次設定 — 上傳所有檔案到 GitHub
第一次使用時點此按鈕，選擇 IPCC 資料夾，把所有靜態檔案推到 GitHub Repository。

⚠️ 危險區域

清除所有資料後無法復原，請謹慎操作。
